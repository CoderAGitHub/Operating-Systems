An operating system can determine the order of tasks that are processed. This is important because some tasks needs to be done first, so the other tasks can function properly. The first task an operating system processes is the computer turning on once the power button is pressed.
Another function of the operating system is to provide a way to establish Internet connections. This is important because people need to use the Internet for research. They can also use the Internet for entertainment. 
One more function of the operating system is to let the user use the applications in the system. This is important because users need some apps to do things. One of the apps could be a game and the user wants to play it.
